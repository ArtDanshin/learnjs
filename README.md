# Как установить

1. `git clone https://github.com/SilverDragoon/learnjs.git`
2. `cd !$`
3. `git checkout geolocation`
3. `npm i`

# Для сброки проекта выполнить
```
gulp build
```
# Для разработки с LiveReload сервером
```
gulp
```
